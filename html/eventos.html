Eventos

HTML tiene la capacidad de permitir que los eventos activen acciones en un navegador.

Los eventos activan acciones en un navegador, como iniciar un JavaScript cuando un usuario hace clic en un elemento.
A continuación se muestran los atributos de eventos globales más comunes que se pueden agregar a los elementos HTML para definir acciones de eventos.

Atributo Descripción
onload Se activa después de que la página termina de cargarse
onresize Se activa cuando se cambia el tamaño de la ventana del navegador
onunload Se activa una vez que se ha descargado una página (o se ha cerrado la ventana del navegador)
onchange Se dispara al momento en que se cambia el valor del elemento
onmouseover Se dispara cuando el puntero del mouse se mueve sobre un elemento
onclick Se dispara con un clic del mouse en el elemento

<button class="btn btn-primary" onclick="alert('¡Hola!')">Da clic</button>

<img class="mt-5" onmouseover="alert('ouhhhh');" src="logo.png" alt="HTML tutorial" >

<body onresize="alert('Nuevo tamaño')">
