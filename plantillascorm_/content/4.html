Tablas

Las tablas HTML permiten a los desarrolladores web organizar los datos en filas y columnas.

La etiqueta <table> define una tabla HTML.
Cada fila de la tabla se define con una etiqueta <tr>.
Cada encabezado de tabla se define con una etiqueta <th>.
Cada celda / datos de la tabla se define con una etiqueta <td>.
De forma predeterminada, el texto de los elementos <th> está en negrita y está centrado.
De forma predeterminada, el texto de los elementos <td> es regular y está alineado a la izquierda.
